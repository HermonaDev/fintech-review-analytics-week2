# Fintech Review Analytics

Analysis of Google Play Store reviews for Ethiopian banking apps.

## Project Structure

- `scripts/` - Data collection and analysis scripts
- `data/` - Raw and processed data files
- `tests/` - Unit tests
- `outputs/` - Analysis results and visualizations
- `docs/` - Documentation and reports

## Banks Analyzed

- Commercial Bank of Ethiopia (CBE)
- Bank of Abyssinia (BOA) 
- Dashen Bank

## Usage

1. Install dependencies: `pip install -r requirements.txt`
2. Run scraping: `python scripts/scrape_reviews.py`
3. Clean data: `python scripts/clean_data.py`
